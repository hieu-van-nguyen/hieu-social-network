1. Prepare your React Application
    npm run build
    # or
    yarn build
2. Create a GitHub Repository:
    Create a new, empty public repository on GitHub.
    Note down the repository name and your GitHub username.
3. Install gh-pages:
    In your React project's root directory, install the gh-pages package as a dev dependency:
    npm install gh-pages --save-dev
    # or
    yarn add gh-pages --dev
4. Configure package.json:
    Open your package.json file.
    Add a homepage property, specifying the URL where your app will be hosted on GitHub Pages:
    "homepage": "https://{your-username}.github.io/{repo-name}",
    Add predeploy and deploy scripts to the scripts section:
    "scripts": {
        "start": "react-scripts start",
        "build": "react-scripts build",
        "test": "react-scripts test",
        "eject": "react-scripts eject",
        "predeploy": "npm run build",
        "deploy": "gh-pages -d build"
    },
5. Initialize Git and Push to GitHub:
    Initialize a Git repository in your project if you haven't already:
        git init
        git add .
        git commit -m "Initial commit"
    Add your GitHub repository as a remote:
        git remote add origin https://github.com/{your-username}/{repo-name}.git
    Push your code to the master (or main) branch on GitHub:
        git push -u origin master
        # or
        git push -u origin main
6. Deploy to GitHub Pages:
    Run the deploy script from your project's root directory:
    npm run deploy
    # or
    yarn deploy
7. Configure GitHub Pages:
    Navigate to your repository on GitHub, Go to Settings > Pages, Under "Source," select the gh-pages branch and the / (root) folder, and Click "Save
